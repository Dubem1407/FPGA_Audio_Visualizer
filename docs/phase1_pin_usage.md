# Phase 1 Pin Usage

This document lists only the pins used in Phase 1 (not the complete DE2-115 CSV).  


---

## Clock Input

| Signal Name | Board Pin | Notes |
|-------------|-----------|--------|
| `clk_27` | PIN_Y2 | 27 MHz onboard clock |

---

## Reset Input

| Signal Name | Board Pin | Notes |
|-------------|-----------|--------|
| `reset_bar` | PIN_M23 | Active-low reset |

---

## Debug Input

| Signal Name | Board Pin | Notes |
|-------------|-----------|--------|
| `debug` | PIN_AB28 | Connected to a switch to select clock rate |
